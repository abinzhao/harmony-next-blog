## 四、性能优化

### （一）代码优化技巧

从 ArkTS 代码层面，介绍优化技巧，如避免不必要的函数调用、合理使用异步操作等。结合代码示例，使用 VitePress 的代码块行高亮功能，突出优化前后的代码差异：

```typescript {3}
// 优化前
function oldFunction() {
  for (let i = 0; i < 1000; i++) {
    // 频繁调用函数
    subFunction();
  }
}

// 优化后
function newFunction() {
  let temp = 0;
  for (let i = 0; i < 1000; i++) {
    // 减少函数调用
    temp += i;
  }
}
```
